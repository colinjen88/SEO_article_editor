# TP 標記編輯器使用手冊

> **版本：** v1.0  
> **更新日期：** 2025-11-04

## 📌 簡介

TP 標記編輯器是一個專為 SEO 文章撰寫設計的簡易編輯工具，提供直觀的雙欄介面，讓您可以邊寫邊看效果。

### ✨ 主要特色

- **雙欄設計**：左側編輯區，右側即時預覽
- **快速插入**：工具列提供常用標記快捷按鈕
- **即時預覽**：輸入自動更新 HTML 效果
- **檔案管理**：支援開啟/儲存 .txt 檔案
- **匯出功能**：一鍵產生完整 HTML 檔案
- **瀏覽器預覽**：查看完整樣式效果

### 2. 快速插入按鈕

點擊工具列的按鈕可快速插入標記：

| 按鈕 | 插入內容 | 說明 |
|------|---------|------|
| **H1** | `(tp_h1)` | 文章主標題 |
| **H2** | `(tp_h2)` | 段落標題 |
| **H3** | `(tp_h3)` | 小標題 |
| **前言** | `(tp_intro)` | 前言區塊 |
| **段落** | `(tp_sec)` | 一般段落 |
| **FAQ** | FAQ 區塊範本 | 完整問答區塊 |

### 3. 基本操作流程

#### 📝 撰寫新文章

1. 點擊 **H1** 按鈕，輸入文章標題
2. 點擊 **前言** 按鈕，撰寫前言內容
3. 點擊 **段落** 按鈕，開始撰寫各段落
4. 使用 **H2**、**H3** 按鈕插入標題
5. 點擊 **FAQ** 按鈕，加入常見問答

#### 💾 儲存檔案

- **儲存**：儲存到目前檔案（Ctrl+S）
- **另存新檔**：儲存到新檔案

#### 👁️ 預覽效果

- **即時預覽**：右側自動顯示（輸入後 0.5 秒更新）
- **手動更新**：點擊「🔄 更新預覽」
- **瀏覽器預覽**：點擊「🌐 瀏覽器預覽」看完整樣式

#### 📤 匯出 HTML

1. 點擊「📤 匯出 HTML」
2. 選擇儲存位置
3. 輸入檔案名稱
4. 點擊儲存

## 🎨 TP 標記語法快速參考

### 基本標記

```
(tp_h1)這是主標題

(tp_intro)
這是前言內容，會被包在特殊區塊中。

(tp_sec)
(tp_h2)這是段落標題
這是段落內容，可以寫很多行。

(tp_h3)這是小標題
這是小標題下的內容。
```

### FAQ 標記

```
(tp_sec_qa)
(tp_h3_q)這是問題一？
(tp_ans)這是問題一的答案。

(tp_h3_q)這是問題二？
(tp_ans)這是問題二的答案。
```

### 完整範例

```
(tp_h1)黃金投資完整指南

(tp_intro)
黃金投資是最穩健的理財方式之一，本文將為您詳細介紹。

(tp_sec)
(tp_h2)為什麼要投資黃金
(tp_h3)保值功能
黃金具有良好的保值功能...

(tp_sec)
(tp_h2)投資方式比較
有以下幾種投資方式...

(tp_sec_qa)
(tp_h3_q)新手適合投資黃金嗎？
(tp_ans)非常適合，因為風險較低...

(tp_h3_q)需要多少資金？
(tp_ans)最低可以從 3,000 元開始...
```

## 💡 使用技巧

### 1. 快速編輯

- 使用快速插入按鈕加速撰寫
- 複製範例文本作為起點
- 善用 Ctrl+Z / Ctrl+Y 復原/重做

### 2. 預覽技巧

- 編輯時不必頻繁手動更新（自動更新）
- 瀏覽器預覽可看到完整 CSS 樣式
- 預覽區可複製 HTML 代碼

### 3. 檔案管理

- 定期儲存避免遺失
- 使用有意義的檔名，如 `黃金投資_20251104.txt`
- 範例檔案在 `input_docs/example_tp_article.txt`

### 4. 匯出 HTML

- 匯出的 HTML 包含完整樣式
- 檔案儲存在 `output/` 目錄
- 可直接用瀏覽器開啟查看

## ❓ 常見問題

### Q: 如何開始撰寫第一篇文章？

**A:** 啟動編輯器後會自動載入範例，可以參考範例格式，或直接清空重新開始。建議先從範例修改，比較容易上手。

### Q: 預覽沒有更新？

**A:** 
1. 檢查是否有輸入錯誤的標記
2. 點擊「🔄 更新預覽」手動更新
3. 預覽更新有 0.5 秒延遲，等待片刻

### Q: 匯出的 HTML 沒有樣式？

**A:** 匯出的 HTML 已包含內建樣式，用瀏覽器開啟應該會有完整效果。如果要使用其他樣式，可以修改匯出功能的 CSS 部分。

### Q: 如何插入圖片或超連結？

**A:** 目前版本主要支援文字內容。如需要圖片或連結，可以：
1. 先用編輯器產生基本結構
2. 匯出 HTML 後手動加入

### Q: 可以同時開啟多個檔案嗎？

**A:** 目前版本一次只能編輯一個檔案。如需編輯多個檔案，可以開啟多個編輯器視窗。

### Q: 儲存的檔案可以在其他工具使用嗎？

**A:** 可以！儲存的是純文字 .txt 檔案，可以：
- 用任何文字編輯器開啟
- 在 SEO Layout GUI 中使用
- 手動轉換為 Word 檔案
