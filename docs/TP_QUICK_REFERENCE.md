# TP 標記快速參考卡

## 🎯 常用標記

```
(tp_h1)     主標題（文章標題）
(tp_h2)     段落標題
(tp_h3)     小標題

(tp_intro)  前言區塊開始
(tp_sec)    一般段落區塊開始

(tp_sec_qa) FAQ 區塊開始
(tp_h3_q)   FAQ 問題
(tp_ans)    FAQ 答案
```

## 📝 基本結構範本

```
(tp_h1)文章標題

(tp_intro)
前言內容...

(tp_sec)
(tp_h2)第一段標題
第一段內容...

(tp_sec)
(tp_h2)第二段標題
第二段內容...

(tp_sec_qa)
(tp_h3_q)問題一？
(tp_ans)答案一...

(tp_h3_q)問題二？
(tp_ans)答案二...
```

## 🚀 快速啟動

```powershell
# 主選單
python main.py

# 直接啟動編輯器
python main.py editor

# 或
python src/tp_editor_gui.py
```

## ⌨️ 快捷鍵

```
Ctrl+S  儲存
Ctrl+O  開啟
Ctrl+Z  復原
Ctrl+Y  重做
```
